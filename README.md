# API - Valoriza
Developed during NLW #06 with NodeJs && Typescript

## Getting started

Clone the project
```
git clone https://github.com/massaaki/NLW-NodeJs-ApiValoriza
```

Install dependences
```
yarn
```

Start server
```
yarn dev
```
